📋 部署步骤（只需4步）

1

点击下方按钮，打开Vercel

跳转到 vercel.com，用GitHub账号登录

2

导入GitHub仓库

点击 Add New → Project，选择仓库 aleilo199607-ctrl/finscreener

3

配置项目根目录

在 Root Directory 一栏填写 frontend（非常重要！）

Framework Preset 会自动识别为 Vite

4

添加环境变量

在 Environment Variables 中添加以下变量，然后点击 Deploy

🔧 需要填写的环境变量

⚠️ 在Vercel的部署页面，展开 Environment Variables，逐条添加以下变量：

变量名（Name） 变量值（Value）
VITE_API_URL https://finscreener-production.up.railway.app
VITE_APP_NAME FinScreener
VITE_APP_VERSION 1.0.0
