# 获取Railway API Token

如果你希望我**完全代替你**执行所有部署步骤，我需要Railway API Token。

## 获取Token的步骤：

### 第一步：登录Railway
1. 访问：https://railway.app
2. 点击 "Login with GitHub"
3. 授权访问

### 第二步：生成API Token
1. 登录后，点击右上角头像
2. 选择 "Settings"
3. 选择 "Tokens" 标签页
4. 点击 "Generate Token" 按钮
5. 输入名称：`FinScreener Deployment`
6. 点击 "Generate"

### 第三步：复制Token
1. 复制生成的Token（一串长字符）
2. **重要**：Token只显示一次，请立即保存

### 第四步：告诉我Token
将Token粘贴在下面的消息中发送给我：

```
我的Railway API Token是: [粘贴你的Token在这里]
```

## 有了Token后，我可以：

1. ✅ 自动创建Railway项目
2. ✅ 自动部署后端代码
3. ✅ 自动配置所有环境变量
4. ✅ 自动获取后端URL
5. ✅ 自动验证部署状态

## 替代方案（不需要Token）：

如果你不想提供Token，也可以：
1. **点击HTML页面中的一键部署按钮**
2. **手动完成部署**（只需点击几次）
3. **然后告诉我你的Railway URL**

## 安全说明：

- Railway API Token有访问你所有项目的权限
- 部署完成后，建议删除或轮换Token
- 不要将Token分享给其他人
- 我只会用这个Token部署FinScreener项目
